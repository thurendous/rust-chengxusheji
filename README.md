# Rust学习项目

这是一个Rust语言学习项目，包含多个示例程序，用于学习Rust的基本概念和语法。

## 项目结构

- `00_hello_world`: Hello World示例
- `01_variables`: 变量和可变性示例
- `02_type`: 数据类型示例
- 更多项目将陆续添加...

## 运行方法

进入任意子项目目录，然后运行：

```bash
cargo run
```
